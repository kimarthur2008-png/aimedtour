# Smart K-Medi

## 🚀 Запуск

```bash
npx live-server
```

Открой http://localhost:8080

---

## 👤 Роли и права

### Обычный пользователь
- ✅ Регистрация (email/пароль или Google)
- ✅ Просмотр больниц, болезней, кейсов, туризма
- ✅ Поиск болезней по keywords
- ✅ Персональный алгоритм подбора клиник (5 шагов) → топ-3 результата (сохраняется в `/algorithmResults`)
- ✅ Отправка заявки на консультацию (форма) → сохраняется в `/consultations`
- ❌ Доступ к админке запрещён

### Администратор (менеджер)
- ✅ Вход в админку `/admin.html` (email/пароль)
- ✅ Просмотр **всех** заявок на консультацию (таблица `/consultations`)
- ✅ Фильтр по статусу: Новые | В работе | Выполнены
- ✅ Поиск по имени/email
- ✅ Смена статуса заявки через дропдаун (new → in-progress → done) → сразу пишет в Firestore
- ✅ Удаление заявки
- ✅ Все изменения видны в реальном времени
- ❌ Редактирование больниц/болезней (нужна отдельная админ-панель FE-10)

## 🔑 Данные Firestore (структура)

Коллекции заполняются автоматически при первом запуске (`seedIfEmpty()` в `app.js`).

| Коллекция | Содержимое | Права |
|-----------|-----------|--------|
| `/hospitals` | 5 больниц (названия, спецх, сертификаты) | все читают, только auth пишут |
| `/diseases` | 8 болезней с keywords для поиска | все читают, только auth пишут |
| `/cases` | 3 кейса пациентов (результаты, testimon) | все читают, только auth пишут |
| `/tourism` | 4 туристических места | все читают, только auth пишут |
| `/consultations` | заявки из форм | все пишут, auth могут писать, только auth читать + удалять |
| `/algorithmResults` | результаты опросника | только auth читают свои, пишут свои |
| `/users` | профили пользователей | только auth (свой профиль) |
| `/settings/main` | счётчики (пациентов, стран, больниц) | все читают, только auth пишут |

## 💬 Быстрые команды

```bash
npx live-server                          # запуск сайта
firebase deploy --only functions        # развернуть Cloud Function
firebase deploy --only firestore:rules  # развернуть Firestore Rules
firebase emulators:start                # локальные эмуляторы (опционально)
```
